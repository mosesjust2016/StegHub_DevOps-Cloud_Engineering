## INSTALL PHP 

* 1. Install PHP - to install 3 packages at once 

```bash
 sudo apt install php libapache2-mod-php php-mysql
```

<img src="../images/install-php.PNG" alt="install php">

* 2. Confirm PHP version installed 

```bash
 php -v 
```
